提示框

用法：

      jQuery().new_notification_window(title,content,color,link,link_blank,decode);

颜色编号：

  1. 灰色
  2. 红色
  3. 绿色
  4. 蓝色
  5. 黄色

decode: 0 or 1 ( 将部分特殊字符转义，比如 <>，1 为开启，不填写则默认为 1 )

link_blank: 0 or 1 ( 新窗口打开链接，1 为开启，不填写则默认为 0 )

小提示块

用法：

      jQuery().new_notification_block(content,color);

颜色编号：

  1. 灰色
  2. 红色
  3. 绿色
  4. 蓝色
  5. 黄色

此处内容的特殊符号必定转义。
